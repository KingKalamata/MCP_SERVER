# MCP Vulnerability Scanner

This project is a vulnerability scanning MCP server that integrates various security tools to perform scans on different targets.

## Included Tools

*   MASScan
*   CVSS Scorer
*   NIST DB for CVE
*   Web scraper for CVEs
*   OpenVAS
*   OWASP ZAP
*   Bandit
*   Wazuh
*   OWASP DefectDojo
*   Nikto
*   Web scraper for vulnerability resolutions
